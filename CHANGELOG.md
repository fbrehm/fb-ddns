# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.1] - 2026-03-26

### Changed

* Moving installation of script manpages into Debian package fb-ddns

## [0.2.0] - 2026-03-25

### Added

* Adding Python files from older version of fb-tools.
* Adding Debian packaging directoy.
* Adding pyproject.toml.
* Adding compile-xlate-msgs.sh, etc/babel.ini, update-env.sh and xtract-xlate-msgs.sh for development tasks.
* Adding directory tests
* Adding translations files in `locale/`.
* Adding CHANGELOG.md.
* Adding tests in `tests/`.
* Adding template.spec for creating RPM packages.
* Adding shared Github workflow for creating Debian, RPM and PyPi packages.

### Changed

* Changed licence from APGLv3.0+ to LGPLv3+

## [0.1.0] - 2026-03-25

### Added
